# Q-Z-Collab Versions

## How to Run

### Current Version (v2 - Dungeon Crawler)
```bash
cd /home/quix/Q-Z-Collab
bun run dev
```

### Run v1 (Original)
```bash
cd /home/quix/Q-Z-Collab/versions/v1
bun install
bun run dev
```

### Run v2 (Dungeon Crawler)
```bash
cd /home/quix/Q-Z-Collab/versions/v2
bun install
bun run dev
```
